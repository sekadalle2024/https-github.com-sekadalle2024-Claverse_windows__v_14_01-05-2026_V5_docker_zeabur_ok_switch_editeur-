# Documentation Export Rapports d'Audit

Ce dossier contient la documentation complète pour les fonctionnalités d'export de rapports d'audit dans Claraverse.

## 📚 Fichiers Disponibles

### Export Synthèse CAC
- **00_COMMENCER_ICI_EXPORT_CAC.txt** - Guide de démarrage rapide
- **GUIDE_EXPORT_SYNTHESE_CAC.md** - Guide utilisateur complet
- **RECAPITULATIF_EXPORT_CAC.md** - Récapitulatif technique
- **exemple de [Export Synthese CAC].doc** - Template de rapport

## 🎯 Fonctionnalités

### 1. Export Synthèse CAC
Génère des rapports structurés au format CAC/Expert-Comptable à partir de:
- Points FRAP (contrôle interne opérationnel)
- Recommandations révision des comptes
- Recommandations contrôle interne comptable

### 2. Export Rapports d'Audit Interne
Génère des rapports d'audit interne structurés:
- Synthèse FRAP
- Rapport provisoire
- Rapport final

## 🚀 Démarrage Rapide

2. Consulter **GUIDE_EXPORT_SYNTHESE_CAC.md** pour les détails
