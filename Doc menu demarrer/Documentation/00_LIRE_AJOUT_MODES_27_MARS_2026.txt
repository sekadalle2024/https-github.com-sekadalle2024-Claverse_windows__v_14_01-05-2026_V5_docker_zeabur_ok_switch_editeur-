═══════════════════════════════════════════════════════════════════════════════

Mode "Methodo audit" pour "Collecte documentaire" :

[Command] = Etape de mission
[Processus] = rapprochements bancaires
[Etape précédente] = Cartographie des risques
[Etape de mission] = Collecte documentaire
[Modele] : Processus, document, Direction, operationnel
[Variable 1] = Contenu de [Variable 1]
[Variable 2] = Contenu de [Variable 2]
[Guide Methodo] : Activate          ← NOUVEAU
[Nb de lignes] = 30

NOTES IMPORTANTES
═══════════════════════════════════════════════════════════════════════════════

⚠️ Les nouveaux modes ont été ajoutés UNIQUEMENT pour les étapes qui utilisent
   [Etape de mission] dans leur commande.

   Couverture, etc.) n'ont PAS été modifiées.

⚠️ Pour E-revision, les modes sont basés sur le mode "normal" car il n'y a pas
   de mode "avancé" (seulement "demo").
