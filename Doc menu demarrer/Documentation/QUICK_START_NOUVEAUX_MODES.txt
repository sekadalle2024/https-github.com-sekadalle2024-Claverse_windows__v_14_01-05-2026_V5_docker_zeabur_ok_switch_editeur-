DÉMARRAGE EN 3 ÉTAPES
═══════════════════════════════════════════════════════════════════════════════

1. DÉMARRER L'APPLICATION
   npm run dev

2. OUVRIR LE MENU DÉMARRER
   - Cliquer sur le bouton "Démarrer" dans le chat
   - Sélectionner E-audit pro ou E-revision

3. TESTER LES NOUVEAUX MODES
   - Choisir une étape de mission
   - Sélectionner "Methodo audit" ou "Guide des commandes"
   - Vérifier que la commande contient la nouvelle variable
