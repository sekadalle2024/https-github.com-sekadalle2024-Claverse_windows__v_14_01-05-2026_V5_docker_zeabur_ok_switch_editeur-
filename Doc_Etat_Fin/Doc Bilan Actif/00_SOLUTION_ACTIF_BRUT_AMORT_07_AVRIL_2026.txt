================================================================================
📊 PRINCIPE DE CALCUL
================================================================================

Pour chaque poste d'actif:

1. BRUT = Somme des comptes classe 2 (hors 28 et 29)
   Exemples: 211 (Frais R&D), 22 (Terrains), 23 (Bâtiments), 24 (Matériel)
   Sens: Débiteur (solde positif)

2. AMORT ET DEPREC = Somme des comptes 28 et 29
   Exemples: 281 (Amortissements), 29 (Provisions)
   Sens: Créditeur (valeur absolue du solde négatif)

3. NET = BRUT - AMORT ET DEPREC
   Valeur comptable nette

TOTALISATIONS:
- AZ: TOTAL ACTIF IMMOBILISE (AA à AS)
- BP: TOTAL ACTIF CIRCULANT (BA à BN)
- BT: TOTAL TRESORERIE-ACTIF (BQ à BS)
- BZ: TOTAL GENERAL (AZ + BP + BT + BU)

================================================================================
📋 FORMAT DE SORTIE
================================================================================

┌─────┬──────────────────────────────┬──────┬──────────┬──────────────┬──────────┬──────────┐
│ REF │ ACTIF                        │ NOTE │ BRUT     │ AMORT DEPREC │ NET      │ NET N-1  │
├─────┼──────────────────────────────┼──────┼──────────┼──────────────┼──────────┼──────────┤
│ AE  │ Frais de R&D                 │  -   │ 1000000  │   200000     │  800000  │    -     │
│ AF  │ Brevets, licences            │  -   │ 5000000  │  1000000     │ 4000000  │    -     │
│ ... │ ...                          │  ... │   ...    │     ...      │   ...    │   ...    │
│ AZ  │ TOTAL ACTIF IMMOBILISE       │  -   │50000000  │ 10000000     │40000000  │    -     │
└─────┴──────────────────────────────┴──────┴──────────┴──────────────┴──────────┴──────────┘

STYLES:
- Lignes normales: Fond blanc
- Lignes de totalisation: Fond bleu clair, bordures bleues
- Sections: Fond bleu très clair
- Montants: Police monospace, alignés à droite
- Valeurs nulles: Affichées comme "-"
