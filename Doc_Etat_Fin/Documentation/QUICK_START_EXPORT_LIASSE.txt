═══════════════════════════════════════════════════════════════════

📋 UTILISATION RAPIDE
═══════════════════════════════════════════════════════════════════

1. Générer les états financiers:
   - Uploader une balance Excel
   - Attendre la génération complète

2. Exporter la liasse:
   - Clic droit sur la table
   - Menu > Traitement Comptable
   - Sélectionner "📋 Exporter Liasse Officielle"
   - OU raccourci: Ctrl+Shift+O

3. Saisir le nom de l'entreprise:
   - Popup s'affiche
   - Entrer le nom (ex: "ENTREPRISE SA")
   - Valider

4. Fichier téléchargé:
   - Nom: Liasse_Officielle_[ENTREPRISE]_[ANNEE].xlsx
   - Emplacement: Dossier Téléchargements
   - Contenu: Liasse remplie avec les valeurs
