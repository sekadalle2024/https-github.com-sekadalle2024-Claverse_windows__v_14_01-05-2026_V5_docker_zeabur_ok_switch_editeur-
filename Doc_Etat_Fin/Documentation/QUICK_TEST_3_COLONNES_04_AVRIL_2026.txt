   EN CAS DE PROBLÈME
═══════════════════════════════════════════════════════════════════════════════

❌ PROBLÈME 1 : Seules 2 colonnes sont affichées

Solution :
   - Vérifier que les modifications ont bien été appliquées
   - Relancer le script de génération
   - Vider le cache du navigateur (Ctrl+F5)

---

❌ PROBLÈME 2 : Colonne N-2 affiche "-" partout

Cause : Le fichier Excel n'a pas d'onglet "Balance N-2"

Solution :
   - C'est normal si le fichier n'a que 2 onglets
   - Utiliser le fichier "P000 -BALANCE DEMO N_N-1_N-2.xls" pour tester

---

❌ PROBLÈME 3 : Erreur Python lors de la génération

Solution :
   - Vérifier que l'environnement conda est activé
   - Vérifier que les fichiers ont bien été modifiés
   - Consulter le message d'erreur complet
