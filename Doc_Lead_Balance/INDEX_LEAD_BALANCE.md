## 📝 Glossaire

### Termes techniques

- **Case 21** : Route dans le router de claraApiService.ts pour Lead Balance
- **SENTINEL_LEAD_BALANCE** : Constante interne pour traitement local
- **claraApiService.ts** : Service principal de gestion des API
- **menu.js** : Script du menu contextuel
- **executeLeadBalance()** : Fonction principale de traitement Lead Balance

### Termes métier

- **Lead Balance** : Analyse comparative de balances comptables N vs N-1
- **Comptes communs** : Comptes présents dans les deux périodes
- **Écart** : Différence entre solde N et solde N-1
- **Accordéon** : Interface pliable pour afficher les résultats
